# rails react template
This is [react_on_rails](https://github.com/shakacode/react_on_rails) + [react-router](https://github.com/ReactTraining/react-router) SPA template.

## stack
- rails
- react_on_rails
- react
- react-router
- redux

## usage

```sh
# in root dir
$ bundle install

# in client/ dir
$ yarn install

# back to root dir
$ foreman start -f Procfile.dev
```
